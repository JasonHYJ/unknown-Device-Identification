# Unknown Device Identification

This repository contains the official implementation of the method proposed in our paper: **"An Unknown IoT Device Identification Method Based on Dual-State Representation Learning and Multi-Task Joint Inference"** . 


## 🔗 Citation

If you find our work useful for your research, please consider citing our paper once it is published. The citation details will be updated here.

## 📂 Repository Structure (Planned)

Once public, the repository is expected to include:
- `dataset/`: Example UK dataset, divided into train, test, and unknown record files.
- `Experiments/`: Scripts for testing some experiments.
- `preProcess/`: Script for preprocessing pcap files to extract features.
- `selfSupervisedProcess/`: Script for self-supervised contrastive learning.
- `supervisedClassification/`: Model training script.
- `testCharts/`: Plotting script.
- `testCharts/`: cripts for various experimental tests.
